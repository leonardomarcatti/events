<h1>📅 Event Manager SPA</h1>
<p>Aplicação Single Page Application (SPA) desenvolvida com React, Node.js e React Router, criada com o objetivo de gerenciar e cadastrar eventos sociais como shows, workshops, reuniões e outros tipos de encontros.
</p>
<p>Este projeto foi desenvolvido principalmente como estudo prático para compreender profundamente os conceitos, funcionamento e arquitetura do React Router, explorando suas funcionalidades de navegação moderna dentro do ecossistema React.</p>
<p>O projeto foi criado como parte de estudos avançados em React, buscando compreender não apenas o uso básico do React Router, mas sua mecânica completa, permitindo a construção de aplicações escaláveis e bem estruturadas.</p>

<h2>🚀 Objetivo do Projeto</h2>
<p>O foco principal deste projeto é o aprendizado e domínio do React Router, incluindo:</p>
<ol>
   <li>Navegação entre páginas sem reload</li>
   <li>Rotas aninhadas (Nested Routes)</li>
   <li>Loaders e Actions</li>
   <li>Manipulação de formulários via rotas</li>
   <li>Redirecionamentos</li>
   <li>Tratamento de erros de rota</li>
   <li>Organização de layouts compartilhados</li>
   <li>Controle de estados baseados em navegação</li>
   <li>A aplicação simula um sistema real de gerenciamento de eventos para aplicar esses conceitos em um cenário prático.</li>
</ol>

<h2>🧩 Funcionalidades</h2>
<ul>
   <li>✅ Cadastro de eventos sociais</li>
   <li>✅ Listagem de eventos</li>
   <li>✅ Visualização de detalhes do evento</li>
   <li>✅ Navegação dinâmica entre páginas</li>
   <li>✅ Estrutura SPA sem recarregamento da página</li>
   <li>✅ Rotas organizadas com React Router</li>
   <li>✅ Layouts reutilizáveis</li>
</ul>

<h2>⚙️ Instalação e Execução</h2>
<ol>
   <li>Clonar o repositório => git clonehttps://github.com/leonardomarcatti/events.git</li>
   <li>Acessar a pasta do projeto</li>
   <li>cd events</li>
   <li>Instalar dependências</li>
   <li>pnpm install</li>
   <li>Executar o projeto</li>
   <li>pnpm run dev</li>
   <li>A aplicação estará disponível em: http://ip:3000</li>
</ol>
